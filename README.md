# MathLibrary Project

Библиотека математических функций на C#

## Описание
Проект содержит библиотеку классов для математических операций:
- Возведение в степень
- Вычисление факториала
- Решение квадратных уравнений

## Примеры использования

### Возведение в степень
```csharp
double result = MathOperations.Power(2, 3); // result = 8
